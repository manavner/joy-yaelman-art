# Joy Yaelman / The Art of Joy — Initial Website Concept

A first Next.js portfolio concept based on public online material for Joy Yaelman / `@joyyaelman`.

## Public references found

- Instagram: https://www.instagram.com/joyyaelman/
- Tumblr archive: https://joyyael.tumblr.com/
- Spoonflower: https://www.spoonflower.com/profiles/the_art_of_joy
- TikTok: https://www.tiktok.com/@joyyaelman

## Current content direction

- Expressive illustration
- Poetry / handwritten image-text feel
- Botanical and intimate visual language
- Pattern design / textile-home surface direction
- Soft handmade portfolio atmosphere

## Important note

The current mockup embeds publicly accessible Tumblr image URLs as a design prototype. Before publishing a production site, confirm image permissions, preferred biography, official portfolio selection, contact details, and commercial goals with Joy.

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build

```bash
NEXT_TELEMETRY_DISABLED=1 npm run build
```
